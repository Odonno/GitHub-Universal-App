# GitHub-Universal-App

A GitHub client as a Universal Windows App.

## Roadmap

### v0.1 (current)

* Authentication of user
* Home page
	* "me" section
	* "activities" section
	* "repositories" section (with search)
	* "users" sections (with search)

### v0.2

* Repository page
	* "description" section
	* "activities" section
	* "code" section
	* "commits" section
	* "issues" section

### v0.3

* User page
	* "info" section
	* "activities" section
	* "repositories" section

### v0.4

* "me" sub-pages
	* followers page
	* following page
	* collaborators page
	* public repos page
	* public gists page
	* private repos page

### v0.5

* Notifications
	* toast notifications (with background tasks)
	* badge notifications
